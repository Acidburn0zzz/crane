Crane is a Docker orchestration tool similar to Docker Compose
with extra features and ultra-fast bind-mounts on Mac.
Check out PRO

Installation

The latest Crane release is 3.5.0 and requires Docker >= 1.13. Please have a look at the changelog when upgrading.

Free version

bash -c "`curl -sL https://raw.githubusercontent.com/michaelsauter/crane/v3.5.0/download.sh`" && \
mv crane /usr/local/bin/crane

PRO version

You will need the key from your purchase:

curl -Lo /usr/local/bin/crane "https://www.crane-orchestration.com/downloads/mac/v3.5.0?key=YOUR-KEY"
chmod +x /usr/local/bin/crane

Now you are ready to use the fast bind-mounts! If you are new to Crane, check out the getting started guide.
